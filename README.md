# @truongbuinh/mcp-test-npm-server

A file and script management MCP server (TypeScript/npm).

> ⚠️ This package contains deliberate security issues for MCPSafe e2e testing.

## Tools

- `read_config(path)` — read a config file
- `delete_file(path)` — delete a file
- `run_script(command)` — run a shell command
- `fetch_data(url)` — fetch data from a URL

## Usage

```bash
npm install @truongbuinh/mcp-test-npm-server
```
